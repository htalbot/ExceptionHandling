Files in this directory:

- ExceptionHandling.mpb

    Files that can be included as a base project in a
    project using ExceptionHandling.

- ExceptionHandling_dependencies.mpb

    This base project file contains the dependencies of ExceptionHandling.
    This file should not be updated manually.

- ExceptionHandling_ex.mpb

    This file contains the specific project MPC definitions.
    This is the common definitions for the type of the project.
